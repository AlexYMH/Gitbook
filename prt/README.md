# NLP预训练发展史

**Contextualized Word Embedding:**

* **Elmo**
* **GPT1.0**
* **ULM-Fit**
* **BERT**
* **GPT2.0**
* **MASS**

**这些模型出现的主要原因在于：普通的word embedding无法区分一个多义词的不同语义，即它没有将一个多义词的不同语义分别编码。**
